# tinyfish-competitor-agent
Autonomous Competitor Intelligence Agent powered by the TinyFish Web Agent API
Autonomous Competitor Intelligence Agent (TinyFish)
This project is built for the TinyFish Hackathon.
It uses the TinyFish Web Agent API to autonomously browse real competitor websites and extract:

Pricing
Product features
Target audience
Positioning
Unique selling points

The agent outputs a clean, structured competitor analysis report, replacing hours of manual research for founders, marketers, and product teams.

🚀 How It Works

Input competitor URLs
TinyFish agent browses each website in real time
Extracts useful content from dynamic pages
Generates a structured summary

This demonstrates a real business workflow automated using autonomous web agents.

🛠 Tech Stack

Python (Google Colab)
TinyFish Web Agent API
Loom (demo recording)
